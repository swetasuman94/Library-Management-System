-Web application for Library Management System.

Languages Used
Node.Js (Version 10.15.1)
MySQL
Jade(Templating language)

How to install
-Install Node.JS from https://nodejs.org/en/
-Go to application location in command prompt.
-Use command `npm install` to install all the dependency.
-Once installation is done, start the application by using the command `npm start`.
-Server will start running on port 8080.
-Open the web application in browser by using localhost:8080 as url.


Application tested on windows operating system.
